# citation docs

メッセージリンクのメッセージをプレビューとして埋め込みに表示するDiscord Bot "citation" の公式ドキュメントです。

Cloudflare Pages でホスティングしています。

- [citation](https://github.com/m2en/citation)

## Usage

- build: `mkdocs build`
- dev: `mkdocs serve`
